# Puzzle Lab — by Rook Foundations

A standalone web app of puzzles and cognitive exercises for children ages 5–12.
It's visually consistent with the main Rook Foundations site (same fonts,
colours, and rook mark) but ships and runs independently.

No accounts, no login, no saved progress — every visit starts fresh.

## Exercises

- **Memory Match** — flip-card pair matching (memory)
- **Pattern Detective** — sequence completion puzzles (logic & pattern recognition)
- **Odd One Out** — visual discrimination grids (focus & attention)
- **Sequence Recall** — Simon-style growing sequence game (working memory)

Each exercise has three difficulty tiers — Explorer (5–7), Thinker (8–9), and
Champion (10–12) — that change grid size, sequence length, or pattern
complexity.

## Development

```bash
npm install
npm run dev      # start local dev server
npm run build    # production build to dist/
npm run preview  # preview the production build
```
